# 🩸 Blood Donation WebApp

A modern, responsive web application to promote and manage blood donations.

## 🚀 Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## 🛠️ Getting Started

```bash
git clone https://github.com/aarthi-kumari/Blood-donation-webapp.git
cd Blood-donation-webapp
npm install
npm run dev
